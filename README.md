# Match Huddle: Real-Time Cricket Fan Engagement Dashboard

A high-fidelity cricket dashboard featuring live score updates, ball-by-ball commentary, real-time fan chat, and interactive voting polls.

## Features
- **Live Score Sync**: Real-time integration with Cricbuzz API via local proxy.
- **Dynamic Commentary**: Event-colored ball-by-ball feed that grows as the match progresses.
- **Interactive Voting**: Prediction polls backed by Firebase Realtime Database.
- **Live Fan Huddle**: Global chat for fans to support their team (RCB, CSK, MI, etc.).
- **Data-Rich UI**: Comprehensive match stats including batting/bowling tables, run rates, and more.

## Tech Stack
- **Frontend**: React, TanStack Router, Vite, Tailwind CSS.
- **Real-Time Database**: Firebase Realtime DB.
- **Animations**: Framer Motion.
- **API Handling**: Axios & Proxy server.

## Getting Started
1. `npm install`
2. `npm run dev`
